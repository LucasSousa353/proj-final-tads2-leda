# proj-final-tads2-leda
Projeto Final da disciplina LEDA do Tecnólogo em Análise e Desenvolvimento de Sistemas no IFSP PTB

Para "rodar" os códigos você deve ter instalado em seu computador a linguagem C++, um compilador e um ambiente de execução do código, como por exemplo o CMD, Replit, VSCode, entre outros.

Realizado na medida do possível por Igor Ramos e Lucas de Sousa por conta da apertada agenda e divergências de escalas de trabalho além de problemas particulares.
